# Edu Toolbox

互动教学工具箱 - 让课堂更有趣，让老师更轻松

## 简介

Edu Toolbox 是一款为教师设计的在线教学工具集合，包含多种互动游戏和实用工具，帮助课堂更加生动有趣。

## 工具列表

| 工具 | 说明 |
|------|------|
| 🎮 五子棋 | 利用五子棋复习同义词和反义词 |
| 🧮 汉字大侦探 | 词语搜索游戏 |
| 🖨️ 汉字大侦探（工作纸） | 可生成词语搜索工作纸 |
| 🎯 生词翻翻看 | 翻转卡牌记忆词汇 |
| 🎮 贪吃蛇 | 吃掉错误单词 |
| 🎮 疯狂的句子 | 词语排序成句 |
| 🔧 随机点名器 | 随机抽取学生 |
持续更新中。。。。。。

## 使用方法

1. 输入邀请码进入
2. 选择分类（Game / Tool / Others）
3. 点击工具卡片开始使用

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS
- GitHub Pages 托管
